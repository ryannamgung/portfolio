Platform Jumper

September 2018

One of my recent projects involved making a jumping game using React and the Phaser library for the frontend, and RAILS postresql for the backend API storage. The end result was a fully functional game where the user could jump from platform to platform, and avoid obstacles to get to a portal. Music Guroo

September 2018

A Ruby of Rails based application with SQLite data storage. The purpose of the site is comfortable scheduling of music lessons. Login & signup functionality for both student and teacher are done using Bcrypt tokens and cookie state preservation. Teachers have the ability to create appointments, and students can book appointments from an available schedules list. Both parties can also edit their profiles.

RSA Cryptography

December 2017

Used Elucids and the RSA prime number algorithm to generate large prime numbers that could be used to form public and private keys. Shared the public keys with the recipient and sent them a message which is turned into a jumble of letters and numbers by the algorithm. Then the recipient can decrypt this message using the key and thus completing the encrypted messaging process.
